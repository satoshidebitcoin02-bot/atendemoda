# Supabase

1. Crie um projeto no Supabase.
2. Abra o SQL Editor.
3. Execute `schema.sql`.
4. Copie a URL e a anon key para `.env.local`.

As politicas RLS deste esqueleto permitem acesso para usuarios autenticados. Em producao, conecte lojas e usuarios em uma tabela de membros antes de liberar clientes reais.
