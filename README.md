# AtendeModa

SaaS para lojistas de roupas que vendem pelo WhatsApp organizarem clientes, atendimentos, vendas e mensagens rápidas.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL

## Como rodar

```bash
npm install
npm run dev
```

Copie `.env.example` para `.env.local` e preencha as credenciais do Supabase.

## Banco de dados

Use o arquivo `supabase/schema.sql` no SQL Editor do Supabase para criar as tabelas iniciais.
